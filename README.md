# VERTEXworkout Admin System

نظام إداري متكامل مصمم لمدربي اللياقة البدنية لإدارة المشتركين، تتبع البيانات الحيوية، وتنسيق الموارد التدريبية.

## المميزات
- **إدارة العملاء:** تسجيل بيانات المشتركين (الاسم، المقاييس الحيوية، الحالة الطبية، والاشتراكات).
- **البحث الذكي:** البحث السريع عن أي مشترك في قاعدة البيانات.
- **إدارة الموارد:** استعراض سريع للبرامج التدريبية، الخطط الغذائية، وقائمة الأدوات الخاصة بـ VERTEXworkout.
- **تتبع البيانات:** حساب مؤشر كتلة الجسم (BMI) تلقائياً عند تسجيل كل عميل.
