# VERTEXworkout

نظام برمجي متكامل لإدارة بيانات العملاء وحساب مؤشرات اللياقة البدنية الأساسية.

## المميزات:
- تسجيل بيانات العميل (الوزن، الطول، العمر، الجنس).
- حساب مؤشر كتلة الجسم (BMI).
- حساب معدل الأيض الأساسي (BMR) لتقدير السعرات الحرارية اليومية.
- حفظ بيانات العملاء تلقائياً في ملف نصي (clients.txt).

## كيفية التشغيل:
1. تأكد من تثبيت Node.js.
2. قم بتثبيت المكتبة اللازمة: `npm install readline-sync`
3. قم بتشغيل البرنامج عبر الأمر: `node index.js`
